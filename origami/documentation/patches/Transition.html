Transition

Takes a number in the range of 0 to 1 and converts it to a number within a range starting with Start Value and ending in End Value.

Transition

  • Progress
  • Start Value
  • End Value
  • Value
Progress

0-1 value that represents how far along a transition you are.

Start Value

The value to output at the start of the transition.

End Value

The value to output at the end of the transition.

Value

A number between your Start and End values
