Browser Chrome

Styles the viewer window to look like a web browser when the composition is running. Click the right most toolbar button to stop the viewer. The back and forward buttons are accessible through the Browser Buttons patch.

Browser Chrome

  • Enable
  • Window Title
  • URL
  • Favicon
  • Enable Back
  • Enable Forward
Enable

Enable or disable the theming of the Viewer window.

Window Title

Set the title of the Viewer window. 10.9 only.

URL

URL to display in the address bar.

Favicon

Takes a 16 px square image and displays it in the address bar. 10.9 only.

Enable Back

Enables or disables the back button in the toolbar.

Enable Forward

Enables or disables the forward button in the toolbar.
