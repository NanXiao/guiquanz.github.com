Pop Animation

Animate a changing value with a spring. You can use the Code Export feature in the Origami menu to use these animations in your built products.

Pop Animation

  • Number
  • Bounciness
  • Speed
  • Progress
Number

The changing value you wish to animate. This is the target value to which Pop Animation will animate.

Bounciness

A qualitative feeling of how springy the animation will feel. This can be changed as the animation is running for advanced effects.

Speed

A qualitative feeling of how fast the animation progresses towards the target value. This can be changed as the animation is running for advanced effects.

Progress

Animated value
